# 人物塑造分析指南

## 1. 核心问题

**人物有没有辨识度？对话能不能遮住名字分清？**

不要复述"主角是某某，做了某某事"，要回答：
- 读者记住的是角色的什么特征？
- 不同角色说话有没有区别？
- 作者用了什么手段让角色"活"起来？

## 2. 主角分析

### 第一次出场印象
- 读者第一次见到主角时，对他有什么印象？
- 是通过什么方式建立的？（动作/语言/选择/遭遇）
- 这个印象是不是能让读者记住他？

### 记忆点手段
作者用了什么手段让读者记住主角？
- 一个标志性动作？（如：验堤时用刀尖挑土）
- 一句有辨识度的话？
- 一个反常识的决定？
- 一个明显的缺陷？

### 核心性格展示
- 主角的核心性格是什么？
- 是通过什么展示的？（对话/动作/选择/他人反应）
- 不是"作者说他是XX样的人"，而是"他做了XX事"

### 说话辨识度
- 主角说话有没有自己的风格？
- 能不能遮住名字分辨出是主角在说话？
- 举出2-3个例子

## 3. 配角分析

### 写得最好的配角
- 哪个配角写得好？好在哪里？
- 有没有单独的辨识度？
- 是不是只在需要时出现（工具人）？
- 能不能补主角短板？

### 对话差异
- 不同角色说话有没有区别？
- 能不能遮住名字分辨出是谁在说话？
- 举出两段对话，说明角色语言差异在哪里

## 4. 反派分析

### 反派出场
- 反派是怎么出场的？
- 第一次出场是不是就让人讨厌？
- 还是先"示好"再暴露？

### 反派人味
- 反派有没有"人味"？还是纯工具人？
- 有没有自己的动机/逻辑/情感？
- 还是只是为主角提供阻力的纸片人？

## 5. 对话差异分析模板

```
角色A：[列举说话特征]
- 用词倾向：[书面/口语/简短/啰嗦]
- 句式特点：[短句多/长句多/反问多/陈述多]
- 语气词：[有无特定口头禅]
- 举例：[具体对话片段]

角色B：[列举说话特征]
- 用词倾向：[与A的对比]
- 句式特点：[与A的对比]
- 语气词：[与A的对比]
- 举例：[具体对话片段]

差异分析：[总结两者的区别]
```

## 6. 退化模式要求

- `full_text`：可精确定位对话特征和行为细节
- `degraded_structure_only`：基于标题和片段推断，降低置信度
- `blocked`：不进行人物分析

## 7. 不合格分析示例

❌ 不合格：
- 主角是个正义的人，他勇敢地对抗邪恶
- 配角张三很忠诚，一直支持主角

✅ 合格：
- 主角第一次出场就在决口夜拒绝签字，用"验过的堤照样决口"直接顶回上官，读者立刻记住他是个"不背锅"的人
- 张三的辨识度在于他每次说话都会先停顿再开口，而且总带着"我也是为了大家"的语气，暗示他的私心
