# Style Signals

## 1. 风格分析原则

- 分析“写法”，不是鉴赏文学性。
- 分析“可迁移信号”，不是评价作者高低。
- 重点找重复出现、稳定有效、能解释读感的特征。

## 2. 句长与段长

观察：
- 短句偏多还是长句偏多
- 一段通常几句
- 大段密集解释是否常见
- 是否适合手机端阅读

建议输出：
- 主趋势一句话
- 例：`以短句推进为主，平均每段 2-4 句，转折时句子更短。`

## 3. 推进手段

重点看哪种手段最常见：
- 对白推进
- 动作推进
- 内心推进
- 旁白说明推进
- 场景切换推进

不要只说“文风流畅”，要指出流畅来自哪种推进方式。

## 4. 表达信号

可观察：
- 常用动词类型：硬动作词 / 轻情绪词 / 说明型动词
- 高频意象：雨、血、门、灯、泥、笑、手、眼等
- 语气偏向：硬、冷、贫、稳、松、狠、闹
- 常见转折手法：短停顿、反问、断句、人物插话

## 5. 对白分析

必须看：
- 对白占比高不高
- 人物说话是否同质化
- 是否有身份感
- 是否通过对白推进信息，而不是堆说明书台词

## 6. 常用词句分析边界

允许：
- 高频词
- 高频短语
- 句式模板
- 意象群

不允许：
- 长句连摘
- 按原文顺序堆语录
- 把原句改几个字就当模板

## 7. 输出建议

### 03_文笔风格.md 推荐结构
- 总体读感
- 句段特征
- 推进方式
- 对白特征
- 风格风险点
- 可迁移建议

### 04_常用词句.md 推荐结构
- 高频动词
- 高频意象
- 高频句式
- 对白口头语
- 适合学的表达习惯
- 不建议学的表达习惯

## 8. 风格风险点示例

- 过度堆砌口头语，后期容易显得油
- 高频反问过多，会显得吵
- 纯对白推进过多，空间感会丢
- 过度依赖同一套章末钩，后期会模板化

## 9. 情绪设计信号（新增）

### 情绪制造方式
观察作者是如何制造情绪的：
- 直接描写情绪："他很愤怒"、"她感到绝望"
- 情境制造情绪：场景、行为、对话、细节组合
- 身体反应：手抖、脸热、心跳加速、冷汗
- 环境映射：天气变化、光线变化、声响暗示

### 情绪密度评估
- 平均每章情绪起伏次数
- 情绪高点/低点的分布
- 有没有整章情绪平淡（说明节奏有问题）
- 情绪密度最高的章节在什么位置

### 章末钩子情绪类型
- 悬念型："接下来会发生什么？"
- 紧张型：停在危机时刻
- 情绪型：停在强烈感受中
- 反转型：刚揭露了一个意外

### 建议输出格式
```
情绪制造方式：主要靠情境制造（场景+动作+对话），很少直接写"他很XX"。
情绪密度：平均每章2-3次情绪起伏，第X章情绪密度最高。
章末钩子偏好：悬念型占60%，紧张型占30%。
风险点：连续多章使用同类型章末钩会导致疲劳。
```
